# Interactive Quiz Website

A modern, interactive quiz application with multiple choice questions, real-time scoring, and grade calculation.

## Features

✅ **10 Multiple Choice Questions** — Test your knowledge across various topics  
✅ **Real-time Score Calculation** — See your score update as you progress  
✅ **Automatic Grading** — Grade assigned based on percentage (A+, A, B, C, D, F)  
✅ **Progress Tracking** — Visual progress bar and question counter  
✅ **Instant Feedback** — Personalized feedback message after completing quiz  
✅ **Responsive Design** — Works seamlessly on desktop, tablet, and mobile  
✅ **Dark Aesthetic Theme** — Easy on the eyes with modern gradient styling  
✅ **Restart Feature** — Retake the quiz to improve your score  

## How to Use

1. Open `index.html` in your web browser
2. Read each question carefully
3. Click an option to select your answer (correct/incorrect answer shown immediately)
4. Click "Next Question" to proceed to the next question
5. After completing all 10 questions, view your results
6. See your score, percentage, grade, and personalized feedback
7. Click "Restart Quiz" to try again

## Grading Scale

- **A+**: 90-100% — Outstanding performance
- **A**: 80-89% — Excellent performance
- **B**: 70-79% — Good performance
- **C**: 60-69% — Satisfactory performance
- **D**: 50-59% — Passing performance
- **F**: Below 50% — Need to study more

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- Uses arrays and loops for question management
- Grade calculation implemented with if-else conditions
- Responsive grid layout for results display
- Smooth animations and transitions
- Modern dark theme with gradient accents

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Gradients)
- Vanilla JavaScript (ES6)
